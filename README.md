# Real-Time Object Detection System 📸

This project is a web-based interface for real-time object detection using a custom model. Built for a dissertation at **Fergana state technical Institute**.

## 🚀 Features
- **Real-time Detection:** Uses TensorFlow.js for browser-based inference.
- **Performance Metrics:** - 🟣 **Inference Time:** Tracks processing speed in ms.
  - 🔵 **Confidence History:** Visualizes model certainty over time.
  - 🟢 **Stability:** Measures detection consistency.
- **UI/UX:** Fully responsive design with **Dark Mode** support.
- **Cross-Platform:** Works on desktop and mobile devices.

## 🛠 Tech Stack
- HTML5 / CSS3 (Inter Font)
- JavaScript (Vanilla)
- [Chart.js](https://www.chartjs.org/) (Data Visualization)
- [TensorFlow.js](https://www.tensorflow.org/js) (Machine Learning)

## 📦 How to Run
1. Clone the repo.
2. Run a local server (e.g., `python -m http.server 8013`).
3. Open `http://localhost:8013` in your browser.
